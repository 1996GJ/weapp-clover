# 四叶草庄园

基于使用[ztaro](https://github.com/FaureWu/ztaro)脚手架快速搭建的电商实战项目`四叶草庄园`

## 已开发的需求

* 项目脚手架搭建
* 登录机制实现

## 计划需求

* 首页
  * 首页资源位
  * 首页品牌推荐
  * 推荐商品
* 商店
  * 品牌列表
  * 品牌商品列表
* 商品详情
* 购物车
* 订单结算，支付
* 我的

  * 客服
  * 关注公众号
  * 新人福利群
* 地址管理
* 订单
